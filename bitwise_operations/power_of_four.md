# Power of Four

**난이도:** Easy  
**문제 링크:** [LeetCode - Power of Four](https://leetcode.com/problems/power-of-four/)

## 문제 설명

정수 `n`이 주어졌을 때, `n`이 4의 거듭제곱(power of four)이면 `true`를, 그렇지 않으면 `false`를 반환하세요.

정수 `n`이 4의 거듭제곱이라는 것은, 어떤 정수 `x`가 존재하여 다음을 만족한다는 의미입니다: `n == 4^x`

## 제약 조건

- `-2³¹ <= n <= 2³¹ - 1`

## 예제

### 예제 1
**입력:** `n = 16`  
**출력:** `true`  
**설명:** `16 = 4²`

### 예제 2
**입력:** `n = 5`  
**출력:** `false`  
**설명:** 5는 4의 거듭제곱이 아닙니다.

### 예제 3
**입력:** `n = 1`  
**출력:** `true`  
**설명:** `1 = 4⁰`

## 솔루션

### 코드
```javascript
/**
 * @param {number} n
 * @return {boolean}
 */
var isPowerOfFour = function (n) {
  return n > 0 && (n & (n - 1)) === 0 && (n & 0x55555555) !== 0;
};
```

### 접근 방법

**비트 조작**을 사용하여 4의 거듭제곱을 판별합니다.

1. `n > 0`: 양수인지 확인합니다.
2. `(n & (n - 1)) === 0`: 2의 거듭제곱인지 확인합니다 (비트에 1이 하나만 있는지).
3. `(n & 0x55555555) !== 0`: 1이 있는 비트 위치가 짝수 위치인지 확인합니다.

**핵심 아이디어:** 4의 거듭제곱은 항상 양수이며, 2의 거듭제곱이면서도 특정 자리(짝수 비트 위치)에만 1이 있는 수입니다. `0x55555555`는 짝수 위치(0, 2, 4, ...)에만 1이 있는 마스크입니다.

### 시간 복잡도
**`O(1)`**

- 비트 연산은 상수 시간입니다.

### 공간 복잡도
**`O(1)`**

- 추가 공간을 사용하지 않습니다.
